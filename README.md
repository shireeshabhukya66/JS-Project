# E-Commerce Online Cake Shop Website

A responsive e-commerce web application developed using HTML, CSS, Bootstrap, and JavaScript. The project allows users to browse cake categories, view product details, and add products to the cart through an interactive and user-friendly interface.

## Features

* Responsive and modern user interface
* Category-based product filtering
* Add-to-cart functionality using JavaScript
* Product detail display
* Client-side form validation
* Mobile-friendly design
* Smooth user experience across devices

## Technologies Used

* HTML5
* CSS3
* Bootstrap
* JavaScript

## Project Objective

The main objective of this project is to improve frontend development skills by building a real-world e-commerce website with responsive design and interactive functionalities.
